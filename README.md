#安装 npm i [包名]


#api

#apiHost: 传入的服务器地址  配置window   eg:http://localhost:9527'
#defaultPhoto  已经有的图片
#showToast  是否显示toast，上传loading
#onChange  图片改变时的方法
#multiple  是否支持多选
#length  单行数量
#total  上传的总数量
#preview  是否允许点击预览
#onPreview  点击图片预览时的回调
